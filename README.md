# Интернет-магазин на ASP.NET Core Razor Pages

**Проект выполнен в рамках производственной практики**  
**Дата завершения:** 23 декабря 2025 г.

## Цель проекта
Разработать рабочий интернет-магазин с адаптивным дизайном для мобильных устройств (720×1440px) и компьютеров (от 1200px).

## Технологический стек
- **Backend:** ASP.NET Core 8.0 Razor Pages
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **База данных:** SQLite (адаптация под macOS, оригинал — SQL Server LocalDB)
- **Аутентификация:** ASP.NET Core Identity с ролями
- **Язык:** C#

## Выполненные требования ТЗ

### Адаптивный дизайн
- Мобильные: фиксированная нижняя навигация, 1 колонка товаров, увеличенные кнопки (≥44px)
- Десктоп: горизонтальная навигация сверху, 4 колонки товаров, боковая панель фильтров

### Обязательные страницы
1. **Главная (/)** — логотип, товары, вход/регистрация или приветствие
2. **Каталог (/Catalog)** — товары с фото, ценой, фильтрация по категориям (только ПК), пагинация, кнопка "В корзину"
3. **Корзина (/Cart)** — список товаров, количество, +/- кнопки, общая сумма, оформление заказа (редирект на логин, если не авторизован), очистка после оформления
4. **Авторизация и регистрация** — стандартные формы Identity
5. **Личный кабинет (/Account/Profile)** — только для авторизованных, заглушка истории заказов
6. **Панель администратора (/Admin)** — только для роли Admin:
   - Список товаров
   - Добавление, редактирование, удаление товаров
